# Student Time Manager 🚀

A beautiful, responsive web app to help students manage their daily tasks and create perfect schedules.

## ✨ Features

- ✅ Add, edit, delete tasks
- 📱 Fully responsive design
- 🕒 Real-time clock & date
- 🎯 Task categories (Study, Exercise, Personal)
- 📊 Productivity statistics
- 💾 Local storage (data persists)
- 🎨 Modern gradient UI
- ⏰ Duration tracking
- 🔍 Filter by category

## 🛠️ Tech Stack

- HTML5
- CSS3 (Gradients, Animations, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Font Awesome Icons

## 🚀 Quick Start

1. Clone/Download the repository
2. Open `index.html` in any modern browser
3. Start managing your tasks!

## 📱 Demo

Live demo: [Deploy to GitHub Pages](https://yourusername.github.io/student-time-manager)

## 🎨 Customization

- Change colors in `css/style.css`
- Add new categories in `js/script.js`
- Modify task fields in the modal
